# Windows Fireworks Screensaver

> 用瀏覽器重現 Windows 經典煙火螢幕保護程式 🎆

一個用 HTML5 Canvas 寫的煙火動畫小作品，模擬螢幕保護程式效果：  
煙火從畫面下方升空，在空中開花綻放，可以調整速度與顏色，當作桌面展示或程式練習都適合。

## ✨ 功能介紹

- 🎇 煙火從畫面底部發射，向上爆開
- 🌈 煙火顏色可調整（主題色配置）
- ⏱ 每個煙火速度不同，看起來更接近真實效果
- 🔁 持續循環播放，適合當「螢幕保護程式風格」背景
- ⏱ 滑鼠點擊可以自製煙火
